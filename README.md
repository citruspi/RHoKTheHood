## RHoK The Hood

Creating a system for displaying Rochester government GIS data.


## Setup

You're expected to already have `npm`, `bower`, and `gulp` installed.

```
$ git clone https://github.com/citruspi/RHoKTheHood.git
$ cd RHoKTheHood
$ bower install
$ npm install
$ make dependencies
$ make
```
